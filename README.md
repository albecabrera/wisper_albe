

# wisper_albe

## Uso

### Compilar y lanzar

```bash
./start.sh
```

Mata el proceso si está corriendo, recompila con los últimos cambios y abre la aplicación.

### Atajo de teclado

**fn + Shift** — enciende y apaga la grabación.

### Signos de puntuación por voz

**Alemán (de-DE)**

| Dices          | Resultado |
|----------------|-----------|
| Komma          | ,         |
| Punkt          | .         |
| Ausrufezeichen | !         |
| Fragezeichen   | ?         |
| Doppelpunkt    | :         |
| Semikolon / Strichpunkt | ; |
| Bindestrich    | -         |
| Absatz         | (párrafo) |
| Neue Zeile / Zeilenumbruch | (línea nueva) |

**Español (es-ES)**

| Dices                      | Resultado |
|----------------------------|-----------|
| coma                       | ,         |
| punto                      | .         |
| signo de exclamación       | !         |
| signo de interrogación     | ?         |
| dos puntos                 | :         |
| punto y coma               | ;         |
| guión                      | -         |
| punto y aparte             | (párrafo) |
| nueva línea                | (línea nueva) |
